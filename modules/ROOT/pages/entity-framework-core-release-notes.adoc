This database provider allows Entity Framework Core to be used with Couchbase Database.
The provider is maintained as part of the https://github.com/couchbaselabs/couchbase-efcore-provider[Couchbase EFCore Project].


[#latest-release]
== Couchbase EF Core 1.0 Releases

We always recommend using the latest version of the Couchbase EF Core Provider -- it contains all of the latest security patches and support for new and upcoming features.
All patch releases for each dot minor release should be API compatible, and safe to upgrade;
any changes to expected behavior are noted in the release notes that follow.
